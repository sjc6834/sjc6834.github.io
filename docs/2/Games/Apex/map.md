# 按键映射

## 键鼠

- 鼠标
    - 左键 开火
    - 右键 按住开镜
    - 中键 标记
    - 滚轮上 前进
    - 滚轮下 跳跃
- 键盘
    - 移动
        - W/A/S/D 移动
        - SPACE 跳跃
        - SHIFT ALT 下蹲
    - 交互
        - F 交互
        - T 地图
        - TAB 背包
    - 武器
        - C/V/X 武器1 武器2 收起武器
        - R 再装填
    - 使用
        - E/Q 小技能 大技能
        - 1/2/3/4/5 小药大药小电大电凤凰治疗包
        - G 手雷
        - H 求生物品
    - 其他
        - Y 按住说话
        - CAPS 标记敌人
        - Z 切换缩放
        - \` 谢谢
        - ENTER 发送信息

## 手柄

已确定：

- L 移动
- R 转动视角
- LB 开镜
- RB 开火
- LT 下蹲
- RT 跳跃
- X 交互 换弹
- \- 地图
- \+ 背包

剩余按键：

- 按下 R
- 按下 L
- 上
- 右
- 下
- 左
- Y
- B（默认）
- A（默认）

剩余功能：

- 按下 R - 拿出武器 收起武器
- 按下 L - 手雷
- 上 - 小药大药小电大电凤凰治疗包（长按切换）
- 右 - // 近战攻击
- 下 - // 角色额外行动
- 左 - 标记（长按切换）
- Y - 小技能
- 左+Y - 大技能
- B - 切换缩放
- A - 切换开火模式（长按装备求生物品）
